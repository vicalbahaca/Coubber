# Landing de planes de suscripcion

Landing estatica con selector de modalidad de pago y tres planes.

Estructura del proyecto:
- index.html
- styles.css
- .gitignore

Como ver en local:
1. Abrir `index.html` en el navegador.
2. (Opcional) Servidor simple: `python3 -m http.server 8080`

Deploy en GitHub Pages:
1. `git init`
2. `git add .`
3. `git commit -m "Init landing"`
4. `git branch -M main`
5. `git remote add origin <URL-del-repo>`
6. `git push -u origin main`
7. En GitHub: Settings > Pages > Deploy from branch `main` y carpeta `/`.
